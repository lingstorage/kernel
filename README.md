# 個人情報ナレッジリポジトリ

このリポジトリは、私個人に関するあらゆる情報（アイデアのメモ、ビジネスプラン、自社の経理関連情報など）を体系的に蓄積するためのハブです。カテゴリは多種多様で、テーマごとにディレクトリを分け、下層で詳細を段階的に開示していきます。

## ディレクトリ概要
- `AGENTS.md`: README運用に関するルール。詳細はこのファイルを参照してください。
- `農業/`: 農業領域の知識・計画・記録をまとめるエリア。栽培ノウハウや作物別の情報を段階的に整理します。

## 利用メモ
- ルールや書式に迷ったら `AGENTS.md` を確認してください。
- 各ディレクトリに README を配置し、進むほど詳細がわかる構成にします（Progressive Disclosure）。
- 情報が増えたら、カテゴリごとにサブディレクトリを追加し、このファイルの構成も適宜更新してください。
